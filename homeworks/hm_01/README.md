# ДЗ_1 Разработка контейнера на Hadoop и выполнение задания

Цель задания:
Разработать контейнер, который работает с поднятым Hadoop и выполняет ряд задач, проверяющих его функциональность.

Используйте Hadoop 3.3.6.
Используйте следующие точки входа:
HDFS: hdfs://master:8020
YARN: master:8032

Внутри контейнера выполните следующие задачи (MAX  кол-во баллов 40):
Создайте директорию на HDFS /createme: 5 баллов.
Удалите директорию на HDFS /delme: 5 баллов.
Создайте файл на HDFS /nonnull.txt с произвольным содержимым: 5 баллов.
Выполните джобу MR wordcount через YARN для файла /shadow.txt: 10 баллов.
Запишите число вхождений слова "Innsmouth" в файл /whataboutinsmouth.txt: 15 баллов.
